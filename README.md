# 아냥식(Anyang Meal)

> *** [앱스토어 링크](https://apps.apple.com/kr/app/%EC%95%84%EB%83%A5%EC%8B%9D/id6466650207)** <br>*** 개발기간 : 2023.09 ~ ing **


## 프로젝트 소개
### **안양대학교의 학식을 편리하게!**
안양대학교의 학생들은 학식을 확인하기 위해 주로 안양대학교의 웹사이트에 직접 들어가 확인해야 했습니다.
이를 개선하기 위해 바로 학식의 정보를 바로 알려줄 수 있는 앱을 만들었습니다.
